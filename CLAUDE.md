# raw-capture-by — Livio Raschle Photography Portfolio

## Project
Photography portfolio website for raw-capture-by.com.

## Structure
- `index.html` — main portfolio (v1, dark/gold)
- `v2/` — editorial v2 variant (light/navy)
- `css/style.css` — v1 styles
- `js/main.js` — v1 scripts (Lenis, Framer Motion, lightbox)
- `images/` — all photos (JPG + WebP)
- `CNAME` — custom domain: raw-capture-by.com

## Deploy
GitHub Pages from `main` branch root.
Custom domain: raw-capture-by.com

## Branch strategy
- `main` — production only, via PR
- `dev` — integration branch
- `feat/*` — new features
- `fix/*` — bugfixes

## Stack
Vanilla HTML/CSS/JS. No build step. Fonts: Raleway (v1), Cormorant Garant + Inter (v2).
